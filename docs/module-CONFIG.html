Module: CONFIG

Initializes, Encapsulates, and Calculates all configurations, provides safe access methods for configs to app and game engine
Stores data in private closure scope: module:CONFIG~settings
Implements an adaptation of Crockford's "Revealing Module Pattern"

(static) init(numLanes, numCols, numEnemies, baseEnemySpeed, gameDifficulty)

Initialize CONFIG object
Parameters:
Name Type Description
numLanes number Number of enemy 'traffic lanes' for game (influences canvas height)
numCols number Total number of columns for game (influences canvas width)
numEnemies number Total number of enemies for game
baseEnemySpeed number Lowest base enemy speed
gameDifficulty number This controls the top speed at which enemies can move as well as the range of possible speeds; Set to any integer between 1 (easy, top speed slow, single-speed) and about 9 (very hard, top speed very fast, 9-speeds). This value can be set higher than 9, but that will quickly make the game too difficult to play.
